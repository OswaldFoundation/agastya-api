Getting Started

Welcome to the getting started tutorial. Here, you'll find how simple it is to install and use Agastya in your website and webapps. Make sure that you have created an account before proceeding with this tutorial.

Installation

Agastya is very easy to install to your webpage. All you need to do is add the Agastya JavaScript library before the end of the </body> tag.

GA-style Code

The preferred method to include Agastya's latest version in your website is to use this script inspired by the Google Analytics script. This will automatically add the Agastya script wherever you have loaded your JavaScript files, <head> or <body>.

<script>!function(a,b,c,d,e){d=a.createElement(b),d.src=c,d.async=1,e=a.getElementsByTagName(b)[0],e.parentNode.insertBefore(d,e)}(document,"script","//cdn.oswald.host/agastya/latest.js");</script>

CORS

You can also use the version-specific library to use secure Cross-Origin Resource Sharing (CORS) features, but you'll have to change your code to recieve updates and new features.

<script src="https://cdn.oswald.host/agastya/0.91/agastya.js" integrity="sha384-q4JFUv8QiFVIPVs3SbFTMDZioniWtmiA/zWIrYgtyIePMQ27pBKcs4UUFUBo0Uut" crossorigin="anonymous" async></script>

Note that CORS is not completely supported in Opera Mini, and IE 10 and below, but that wouldn't cause any loading problems.

Initialization

Your users can toggle Agastya by pressing the keyboard combination F + J, the signifier home keys users can feel. For mouse users, you have to create a button with the data-agastya attribute.

<button data-agastya>Accessibility</button>

Agastya finds all elements with the attribute, but it is preferred to use a <button> tag or use the ARIA role="button" and add the necessary key binding to the element you're using.

Analytics

After you've installed and tested Agastya, you can log in to your dashboard and access usage analytics.

Methods

Agastya features a powerful API with a number of methods you can call.

Open

As the name suggests, you can use this method to open the Agastya widget. If it is already open, there will be no effect to calling this method. When a user opens Agastya the first time, the widget stylesheet is loaded, followed by the widget options, and finally additionally required scripts.

agastya.open();

You can add this method to a button with a simple onclick event, for example:

<button onclick="agastya.open()">Open Accessibility Options</button>

Close

Similarly, you can use this method to close the Agastya widget. If it is already closed, there will be no effect to calling this method. When a user closes Agastya, the widget is not removed from the DOM, it just fades out, and is displayed again when the open method is called.

agastya.close();

You can add this method to a button with a simple onclick event, for example:

<button onclick="agastya.close()">Close Accessibility Options</button>

Toggle

This method is the most efficient way to open and close the widget using just one button. It opens the widget if it is closed and vice versa.

agastya.toggle();

You can add this method to a button with a simple onclick event, for example:

<button onclick="agastya.toggle()">Accessibility Options</button>

Font Size

This method is used to change the font size of the webpage. Using this, users can increase the font size for a more comfortable reading experience. It is especially useful for audiences with slight visual impairment, like senior citizens.

agastya.fontSize(parameter);

The parameters allowed are:

Parameter Code Description
"increase" 
agastya.fontSize("increase")
 Increases the font size by 10%.

You can add this method to a button with a simple onclick event, for example:

<button onclick="agastya.toggle()">Accessibility Options</button>
